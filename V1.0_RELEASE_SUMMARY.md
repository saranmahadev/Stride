## Installation & Usage

### Install from Source
```bash
git clone https://github.com/saranmahadev/Stride.git
cd Stride
pip install -e .
```

### Quick Start
```bash
# Initialize project
stride init

# Select AI agents interactively
# (Cursor, Claude, Windsurf, etc.)

# Use agent commands in your AI tool
# Example: /stride:init

# Monitor from terminal
stride list
stride status
stride show SPRINT-XXXXX
```

## Known Limitations

### Not Implemented (Future Roadmap)
1. Advanced configuration management (config.yaml)
2. File manager utilities (direct operations in commands)
3. Database backend (file-based only)
4. Web UI (CLI only)
5. CI/CD integration templates
6. Plugin system (built-in agents only)

### Design Decisions
- **File-based storage**: Simple, version control friendly
- **Placeholder modules**: Documented for future extensibility
- **No web UI**: Focused on CLI and agent integration
- **No database**: Markdown files serve as data store
